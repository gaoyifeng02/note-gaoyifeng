## 八、项目 DDD 分层结构

```
mcp-gateway (parent pom, groupId: com.gaoyifeng)
├── mcp-gateway-api              # 接口定义层 (对外契约)
├── mcp-gateway-app              # 启动模块 (Spring Boot Application)
├── mcp-gateway-case             # 用例编排层
├── mcp-gateway-domain           # 领域层 (核心业务逻辑)
├── mcp-gateway-infrastructure   # 基础设施层 (MyBatis DAO, Retrofit 调用)
├── mcp-gateway-trigger          # 触发器层 (HTTP Controller)
└── mcp-gateway-types            # 通用类型、常量、异常定义
```

**模块依赖关系：**

```
trigger -> domain, infrastructure, types, api, case
app -> trigger, infrastructure
infrastructure -> domain
domain -> types
case -> domain
api -> (独立，仅依赖 lombok, validation, reactor)
types -> (独立，仅依赖 spring-boot-starter-web, lombok)
```

## 九、环境 Profile

| Profile | JVM 堆内存 | 用途 |
|---|---|---|
| dev (默认) | -Xms1G -Xmx1G | 开发环境 |
| test | -Xms1G -Xmx1G | 测试环境 |
| prod | -Xms6G -Xmx6G | 生产环境 |

所有 Profile 统一使用 **G1 GC** (`-XX:+UseG1GC`)。
