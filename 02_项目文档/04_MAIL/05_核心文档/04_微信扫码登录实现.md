## 2. 关键实现细节

### 2.1 二维码创建（ILoginPort）

```java
// LoginPort.java - infrastructure 层
@Component
public class LoginPort implements ILoginPort {

    @Resource
    private IWeixinApiService weixinApiService;  // Retrofit2 接口
    @Value("${weixin.config.app-id}")
    private String appid;
    @Value("${weixin.config.app-secret}")
    private String appSecret;

    @Override
    public String createQrCodeTicket() {
        // 1. 获取 access_token
        WeixinTokenResponseDTO token = weixinApiService.getToken(
            "client_credential", appid, appSecret);

        // 2. 创建带参数的临时二维码
        WeixinQrCodeRequestDTO request = WeixinQrCodeRequestDTO.builder()
            .expireSeconds(2592000)     // 30天有效
            .actionName("QR_STR_SCENE")
            .actionInfo(new WeixinQrCodeRequestDTO.ActionInfo(
                new WeixinQrCodeRequestDTO.Scene(
                    new WeixinQrCodeRequestDTO.SceneStr(
                        UUID.randomUUID().toString()))))
            .build();

        // 3. 返回 ticket
        WeixinQrCodeResponseDTO response =
            weixinApiService.createQrCode(token.getAccess_token(), request);
        return response.getTicket();
    }
}
```

### 2.2 扫码回调处理

```java
// WeixinPortalController.post()
@PostMapping(value = "receive", produces = "application/xml; charset=UTF-8")
public String post(@RequestBody String requestBody, ...) {
    // 1. XML 消息解析
    MessageTextEntity message = XmlUtil.xmlToBean(requestBody, MessageTextEntity.class);

    // 2. 判断扫码事件
    if ("event".equals(message.getMsgType()) && "SCAN".equals(message.getEvent())) {
        // 3. 保存登录状态
        loginService.saveLoginState(message.getTicket(), openid);
        return buildMessageTextEntity(openid, "登录成功");
    }

    return buildMessageTextEntity(openid, "测试本案例，需要请扫码登录！");
}
```

### 2.3 登录状态管理

```java
// WeixinLoginService.java
@Service
public class WeixinLoginService implements ILoginService {

    @Resource
    private Cache<String, String> openidToken;  // Guava Cache

    @Override
    public void saveLoginState(String ticket, String openid) {
        openidToken.put(ticket, openid);    // 写入 Cache
        loginPort.sendLoginTempleteMessage(openid);  // 发送模板消息
    }

    @Override
    public String checkLogin(String ticket) {
        return openidToken.getIfPresent(ticket);  // 查询 Cache
    }
}
```

## 3. 微信 API 对接

### 3.1 Retrofit2 接口定义

```java
// IWeixinApiService.java
public interface IWeixinApiService {

    @GET("/cgi-bin/token")
    WeixinTokenResponseDTO getToken(
        @Query("grant_type") String grantType,
        @Query("appid") String appid,
        @Query("secret") String secret);

    @POST("/cgi-bin/qrcode/create")
    WeixinQrCodeResponseDTO createQrCode(
        @Query("access_token") String accessToken,
        @Body WeixinQrCodeRequestDTO request);

    @POST("/cgi-bin/message/template/send")
    void sendTemplateMessage(
        @Query("access_token") String accessToken,
        @Body WeixinTemplateMessageDTO request);
}
```

### 3.2 签名验证

微信公众号验签使用 SHA1 算法：

```java
// SignatureUtil.java
public static boolean check(String token, String signature, String timestamp, String nonce) {
    String[] arr = new String[]{token, timestamp, nonce};
    Arrays.sort(arr);
    StringBuilder sb = new StringBuilder();
    for (String s : arr) {
        sb.append(s);
    }
    String tmp = DigestUtils.sha1Hex(sb.toString());
    return tmp.equals(signature);
}
```

## 4. 安全注意事项

| 项目 | 说明 |
|------|------|
| 验签 token | 当前硬编码为 "b8b6"，生产环境应从配置读取 |
| access_token | 每次请求都获取新的，生产环境应缓存（有效期2小时） |
| openid 存储 | 当前使用 Guava Cache 内存存储，重启丢失，生产环境应持久化 |
| JWT 生成 | 代码中引用了 JWT 依赖，实际未使用，注释说明"openid 可以生成 jwt token" |
