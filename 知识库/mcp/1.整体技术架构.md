AI MCP Gateway 是一个基于 Model Context Protocol (MCP) 的智能网关服务系统，专为 AI 智能体应用提供高性能、可扩展的 MCP 实时通信基础设施。解决 AI 应用与各种数据源、工具和服务之间的连接复杂性，提供统一的接口和协议标准。

整个实现过程是以 MCP 协议 JSON-RPC2 通信结构，设计实现流式传输 Streamable HTTP 框架，完成 MCP 网关接口协议解析和消息处理。

在消息处理过程中，则对注册进来的 http/rpc/... 接口进行调用。这部分会包括接口的描述性信息获取以及会话执行调用。
