### 包结构

新建项目时，每个包都要创建，每个包下需要创建空文件（temp.txt）用于占位。

- **{project}-api/**
    - **dto/** # 数据传输对象
        - **{业务}/** # 按业务拆分
    - **response/** # 统一响应封装
    - 业务接口 IXxxService
- **{project}-app/**
    - **config/** # 基础设施配置
    - xxxApplication（项目名）
    - **test/** 集成测试
    - resources 资源
- **{project}-domain/**
    - **{子域}/**
        - **adapter/**
            - **port/** # 防腐层端口接口
            - **repository/** # 仓储接口
        - **event/** # 领域事件
        - **model/**
            - **aggregate/** # 聚合根
            - **entity/** # 实体
            - **valobj/** # 值对象
        - **service/** # 领域服务
- **{project}-infrastructure/**
    - **adapter/**
        - **port/** # 端口实现（调用外部接口）
        - **repository/** # 仓储实现（持久化）
    - **dao/**
        - **po/** # 持久化对象
    - **gateway/**
        - **dto/** # 外部接口数据传输对象
    - **redis/** # 缓存服务
- **{project}-trigger/**
    - **http/** # HTTP 控制器
    - **job/** # 定时任务
    - **listener/** # 事件监听
- **{project}-types/**
    - **common/** # 全局常量
    - **enums/** # 枚举定义
    - **event/** # 事件基类
    - **exception/** # 统一异常
    - **sdk/**
        - **{第三方服务}/** # 按第三方服务拆分
