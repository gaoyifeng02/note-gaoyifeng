 1. asset_account - 用户资产总览表 (满足基本要求)

  CREATE TABLE `asset_account` (
    `id` BIGINT UNSIGNED NOT NULL AUTO_INCREMENT COMMENT '自增ID',
    `user_id` VARCHAR(32) NOT NULL COMMENT '用户ID',
    `total_balance` DECIMAL(12,2) NOT NULL DEFAULT 0.00 COMMENT '总现金余额',
    `create_time` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    `update_time` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    PRIMARY KEY (`id`),
    UNIQUE KEY `uq_user_id` (`user_id`)
  ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='用户资产总览表';

我准备做个资产管理系统，功能如下：
1.可以导入自己的月收入。
	1.设置月收入xxx
	2.设置发薪日时间，到达发薪日自动计算流水。（月、日）
2.可以导入自己的现有余额。
3.可以导入自己在某一天的突发收入（例如奖金、炒股等）。
4.可以导入自己的欠款，借贷类型，xx元，待还款xx月。还款日期。
5.可以导入固定消费类型，房租、房贷、饮食，固定消费类型也可以指定期限。
6.可以导入自己的存款计划和存款利率，如当前存款产品利率，计划每月存入多少，当前已有多少。
7.可以导入突发一次性欠款，可以分多次慢慢去删减这个欠款。
8.要有总览页面可以展示，要求可以很只管的看出自己的资产变化来进行管理，可以看出来自己的资产变化趋势，比如总资产，存款，剩余款，负债等。
9.有一键计算功能，在当前情况下，如果要还清所有负债需要多久，当前情况下，实现xx需要多久。

备注：
1.用户固定收入（可以设置多个薪水和发薪日，这种就是相当于固定收入），可以设置触发时间（按周、按月、按年、按日、一周的某几日）
