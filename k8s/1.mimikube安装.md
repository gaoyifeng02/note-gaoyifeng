### 安装

minikube安装文档：https://kubernetes.io/zh-cn/docs/tutorials/hello-minikube/

安装后设置环境变量：C:\Program Files\Kubernetes\Minikube 

启动：

minikube start

需要windows有docker环境！

windows安装docker环境：https://blog.csdn.net/qq_41644183/article/details/132305487

### 使用：

暂停 Kubernetes 而不影响已部署的应用程序：

```shell
minikube pause
```

恢复已暂停的实例：

```shell
minikube unpause
```

停止集群：

```shell
minikube stop
```

更改默认内存限制（需要重启）：

```shell
minikube config set memory 9001
```

浏览易于安装的 Kubernetes 服务目录：

```shell
minikube addons list
```

创建第二个集群，运行旧版本的 Kubernetes：

```shell
minikube start -p aged --kubernetes-version=v1.34.0
```

删除所有minikube集群：

```shell
minikube delete --all
```
